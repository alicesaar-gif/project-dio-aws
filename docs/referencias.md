# Documentação de Referência – Serviços AWS

Documentação oficial da AWS utilizada como base técnica do projeto.

## Amazon S3

Serviço de armazenamento de objetos escalável, com suporte a Intelligent-Tiering e políticas de lifecycle.

https://docs.aws.amazon.com/s3/

## AWS Lambda

Plataforma serverless para execução de funções orientadas a eventos, sem provisionamento de servidores.

https://docs.aws.amazon.com/lambda/

## AWS Budgets

Ferramenta de controle financeiro da AWS para monitoramento de gastos e criação de alertas orçamentários.

https://docs.aws.amazon.com/cost-management/latest/userguide/budgets-managing-costs.html
